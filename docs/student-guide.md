# SelfPass 学生使用指南 / Student Guide

---

## 中文版

### 设计理念

SelfPass 基于**布鲁姆精熟学习理论（Bloom's Mastery Learning）**：你必须真正掌握当前知识点才能继续前进。我们不鼓励盲目跳进度——每一步都扎实，下一步才稳。

### 学习流程

1. **注册登录**：访问网站 → 注册学生账号 → 加入班级（输入老师给的邀请码）
2. **选择学科**：在仪表盘主页选择你要学习的学科（数学、物理、化学等）
3. **进入课程**：点击你感兴趣的课程，查看知识点和教学视频
4. **过关测试**：每个课时都有通关测试。规则如下：
   - 连续答对 7 题 → ✅ 通关
   - 或答题 ≥ 10 题且正确率 ≥ 90% → ✅ 通关
   - 累计答错 3 题 → ❌ 失败，冷却 10 分钟后重试
5. **查看进度**：仪表盘顶部有进度条，每通过一课都会更新

### 错题本

- 答错的题自动进入错题本，按课程-章节分类
- 点击 **🎯 错题重测** 进入闯关模式重做错题
- 答对自动标记 ✓ 已掌握，答错标 🌶️ 反复错误
- 每题都可以点 **🧠 AI 生成知识点** 查看相关知识总结
- 支持导出 PDF 打印

### 收藏课程

课程卡片右上角 ☆ 点一下变 ⭐，收藏的课程会出现在仪表盘「我的收藏」区域。

### 常见问题

**Q: 测试失败了怎么办？**
冷却 10 分钟后可以重新闯关。利用这段时间去知识树看视频学习。

**Q: 怎么加入班级？**
仪表盘左侧有「输入班级邀请码」的输入框，找老师要邀请码。

## English Version

### Design Philosophy

SelfPass is built on **Bloom's Mastery Learning Theory**: you must fully master the current material before advancing. No skipping, no guesswork — every step solid before the next.

### Learning Flow

1. **Sign Up**: Register as a student → join a class using the teacher's invite code
2. **Choose a Subject**: Select from 8 subjects on your dashboard (Math, Chemistry, etc.)
3. **Enter a Course**: View knowledge points and video resources
4. **Gate Test**: Each lesson has a mastery test:
   - 7 consecutive correct → ✅ Pass
   - ≥ 10 questions with ≥ 90% accuracy → ✅ Pass
   - 3 total wrong → ❌ Locked for 10 minutes
5. **Track Progress**: Progress bar at the top of the dashboard updates after each pass

### Wrong Question Book

- Wrong answers are automatically collected, organized by course and chapter
- Click **🎯 Wrong Question Retest** to re-attempt in test mode
- Correct answers → ✓ Mastered; Wrong again → 🌶️ Repeated Error
- **🧠 AI Generate Knowledge Point** on any question for a full concept breakdown
- Export to PDF for printing

### Favorite Courses

Click ☆ on any course card to favorite it (⭐). Favorited courses appear on your dashboard for quick access.

### FAQ

**Q: What if I fail a test?**
After a 10-minute cooldown, you can retry. Use the time to review.

**Q: How do I join a class?**
Enter the invite code from your teacher in the input field on the dashboard.
